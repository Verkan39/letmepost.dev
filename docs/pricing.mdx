## Pricing principles

- **Flat rate, not per-profile.** The most-cited complaint about every incumbent is per-profile or per-seat pricing. We don't tax you for connecting more accounts.
- **A real free tier, not a trial.** Persistent, indefinite, sized so an indie dev or hobby project never has to upgrade.
- **Self-host stays free, forever.** The OSS image is the same code as the hosted SaaS. No feature gate, no open-core trick. [Self-host docs](/self-host/quick-start).
- **No contracts, no auto-renewal abuse.** Month-to-month, cancel anytime, downgrade anytime.

## Tiers

Pricing is per org, not per account. The number on the card is what your whole org pays.

| tier | price | what you get |
|---|---|---|
| **Free** | $0 / mo | 50 posts / month. All 7 platforms, all API surface. MCP server + CLI. Community Discord support. |
| **Pro** | $79 / mo | 5,000 posts / month. All 7 platforms, all API surface. MCP server + CLI. All webhook events. 30-day publish logs. Email support, 24h response. |
| **Business** | $299 / mo | 25,000 posts / month. All 7 platforms, all API surface. MCP server + CLI. White-label OAuth. 99.9% uptime SLA. 180-day publish logs. Priority Slack + email, 4h response. |
| **Self-host** | $0 / mo | Same code as hosted. Bring your own Postgres + Redis. No license server, no telemetry. |

Annual billing saves 2 months on Pro when it turns on.

<Note>
**Founder pricing.** $40/mo locked for life on Pro if you sign up in the next 30 days. Normally $79/mo. Forfeit on cancel.
</Note>

Need more volume, SSO, a custom SLA, or a DPA? Email us and we'll set up a custom plan. No marketplace tier-stacking games.

We intentionally start with two tiers. Once we have real usage data we'll add tiers where the breakpoints actually are. More tiers right now is just decision friction for you and complexity for us.

## What's actually live

- **Bluesky**, **X**, and **Pinterest** are live for every signup today.
- **Instagram**, **LinkedIn**, **Facebook Pages**, and **Threads** are gated on per-platform developer-portal reviews. Each ships as the review clears. Same API, same error envelope.
- Live status is queryable at `GET /v1/platform-versions` and tracked in the [changelog](/changelog).

## What's free during alpha

Everything. While we're in alpha, the hosted SaaS is free for everyone, there's no quota, no rate-limit ceiling beyond what's enforced for abuse prevention, and no card on file.

In return, you should expect:

- **No SLA today.** We're stable enough for production workflows but not promising one.
- **Migration help when billing turns on.** You'll know the prices, get advance notice, and have time to decide. Any workflow that connected during alpha gets a 60-day grace window after announcement before charges begin.
- **Approval timing is upstream.** Most of our platforms gate launch on their own developer-portal reviews. If your business depends on a specific platform that's still in review, self-host with your own credentials is the immediate path.

## What we don't charge for

- **Connected accounts.** Connect every platform, every account, every Page. No per-row tax.
- **Profiles.** A single org can carve into N profiles for per-client isolation. Profiles are free.
- **Team members.** No per-seat ladder.
- **Webhook endpoints.** Register as many as you need, we'll deliver to all of them.
- **API keys.** Mint a key per service, per script, per cron. Keys are free.
- **Webhook deliveries / retries.** Built into the contract, not a billable line item.
