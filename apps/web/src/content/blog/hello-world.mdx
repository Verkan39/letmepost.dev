MDX gives you Markdown plus Astro components inline. Code blocks render
with the same theme as the marketing site:

Inline `code` works. **Bold** and *italic*. Links land at the
[brand color](https://letmepost.dev). Lists, tables, blockquotes —
all rendered through the `prose-blog` styles in the post layout.

## How to publish

1. Drop a new `.mdx` file in `src/content/blog/`. The file name
   becomes the URL slug (`my-post.mdx` → `/blog/my-post`).
2. Fill out the frontmatter — `title`, `description`, `pubDate` are
   required. The schema lives in `src/content.config.ts`.
3. Set `draft: false` (or remove the line) when ready to ship.
4. Push. Sitemap, RSS, and per-post structured data update on deploy.
